# PACT

**Pull And Check Tools** is a compact raid control panel for WoW Classic Anniversary / TBC Classic Anniversary.

PACT keeps common raid-leader controls close to your cursor without keeping the full Blizzard Raid Manager open.

## Features

- Compact draggable mini panel.
- Ready Check, Pull timer, Role Check, Break timer, and optional Cancel button.
- Break timer support using BigWigs-style `/break` behavior when available.
- Pull and Break controls wait for an active Role Check to finish.
- Horizontal or vertical layout.
- Optional reverse button order.
- Configurable panel scale.
- Solo display mode for positioning the panel outside a group or raid.
- Option to hide the Blizzard Raid Manager.
- Per-character saved settings.

## Commands

```text
/pact
/pact lock
/pact unlock
/pact reset
/pact blizzard
```

The configuration panel is also available from the in-game addon options.

## Localization

PACT supports English, Brazilian Portuguese, German, Spanish, French, Italian, Korean, Russian, Simplified Chinese, and Traditional Chinese clients.

## Installation

Install through CurseForge or download `PACT.zip` from the latest GitHub Release. The installed addon folder must be:

```text
World of Warcraft/_anniversary_/Interface/AddOns/PACT/
```

Do not install from GitHub's green **Code > Download ZIP** button. That file is a source snapshot, not the packaged addon.
